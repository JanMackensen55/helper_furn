Class: MainScene

MainScene

This is the main scene of the game. Here every game object is displayed as well as the map, the buildings, the Player. This scene also handels the timing, the camera to enable movement and it stores the profits of the player inside a Soci survey variable if activated.

objective number the total optimal profit for the year.

nextMonth()

This function is triggered if the month should be switched. it steps the monthIterator to the next month, maintains the profits of the player, resets the available time of the workshops and starts a new timer for the new month that expires to start a new month. If the last month is over, this function also triggers a final summary that informs the player that the game is over.
