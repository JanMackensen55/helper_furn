Class: DialogueSchedule

DialogueSchedule

This class takes care of the dialogues that appear during the game. Dialogues inside this structure are displayed after each other to prevent dialogues from appearing at the same time. The dialogues that are added to the scheduler are organized as a list, that will be worked off item per item.
