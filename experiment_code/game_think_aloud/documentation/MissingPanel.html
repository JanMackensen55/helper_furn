Class: MissingPanel

MissingPanel

This Class creates a panel to inform the player of missing material. It is used to be placed behind a card and appear if the player tries to drag the slider to a position the player cannot afford.

new MissingPanel(data)

Parameters:
Name Type Description
data object the data object used to create the panel.
Properties
Name Type Description
parent Phaser.GameObject a gameObject the panel should be placed behind. This is usually a Card.
x number the x coordinate where the panel should be placed.
y number the y coordinate where the panel should be placed.
scalingX number specifies how much the sprites are scaled in x direction
scalingY number specifies how much the sprites are scaled in y direction
