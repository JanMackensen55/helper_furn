Class: ProgressBar

ProgressBar

This class implements an horizontal progress bar that can be used to show item usage or time left.

new ProgressBar(data)

Parameters:
Name Type Description
data object the data which is required to represent a progressbar.
Properties
Name Type Description
scene Phaser.Scene the scene where the progressbar should be rendered.
x number the x coordinate of the progress bar.
y number the y coordinate of the progress bar.
workshop string the name of the workshop or material that is used. It will be displayed inside the progressbar.
image Phaser.Sprite the image object, that will be displayed to the left of the progress bar. (see Phaser documentation).
scalingX number specifies how much the sprites are scaled in x direction
scalingY number specifies how much the sprites are scaled in y direction
