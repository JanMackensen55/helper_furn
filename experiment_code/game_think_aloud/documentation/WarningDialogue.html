Class: WarningDialogue

WarningDialogue

This class implements a dialogue, that is shown if the player finishes a month without bulding something. It lets the player decide if he really wants to finish the month or if he wants to build something

new WarningDialogue(data)

Parameters:
Name Type Description
data object the data object, used to create the dialogue.

Name Type Description
scene Phaser.Scene the scene where the dialogue is placed.
parentScene Phaser.Scene The Scene in which the Scene in which the Dialogue is placed is placed (MainScene)

y number the y coordinate of the dialogue.
scalingX number factor with which the scene is scaled on the x axis
