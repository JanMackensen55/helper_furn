<inner> createArrow(scene, oldPrice, newPrice, x, y)

Decides which arrow should be displayed and where the arrow should be displayed. If oldPrice is higher than newPrice, the arrow will be red and downwards, vice versa the arrow will be green and upwards. If they are the same, a white line will indicate that no change has happened.
Parameters:
Name Type Description
scene Phaser.Scene the scene, to render the symbols
oldPrice number the old price for example the price of an item one month before.
newPrice number the new price to be compared to the old price.
x number the x position where the arrow should be rendered.
y number the y position where the arrow should be rendered.

the symbol depending on the old and new price. If old and new are the same the symbol will be a line, if the old price is higher than the new price, then the arrow is red pointing down. Vice versa the arrow will be green and point up.
