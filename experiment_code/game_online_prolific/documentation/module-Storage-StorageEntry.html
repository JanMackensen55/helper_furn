Class: StorageEntry

Storage~ StorageEntry

Creates an entry for an item in the storage. It shows which parts the item consists of, how many of the parts are currently in the storage and how many parts are needed to build the item.

new StorageEntry()

Parameters:
Name Type Description
data.scene Phaser.Scene the Phaser scene where the StorageEntry should be displayed.
data.x Number Position of the StorageEntry on the x axis.
data.y Number Position of the StorageEntry on the y axis.
data.item Array item for which an entry is to be created
