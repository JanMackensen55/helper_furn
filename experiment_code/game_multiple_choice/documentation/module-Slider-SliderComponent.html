Class: SliderComponent

Slider~ SliderComponent

This class implements the slider logic. It takes care of the scaling of values and the calculation of the drag position.

new SliderComponent(data)

Parameters:
Name Type Description
data object the data object that is used to create a slider component.
Properties
Name Type Description
scene Phaser.Scene the scene to render the component in.
card Card
x Number the x coordinate of the slider component.
y Number the y coordinate of the slider component.
maxValues Number this number will be displayed, if the slider is dragged to the rightmost position. All values inbetween 0 and maxValues will be scaled according to this value.
scalingX number specifies how much the sprites are scaled in x direction
scalingY number specifies how much the sprites are scaled in y direction

scaleSliderVal(x, min, max, factor)

This function converts the coordinates of the slider knob to a corresponding item quantity, depending on the scaling. If for example the maximum value for a slider is set to 10, that means that if it is dragged to the center, the value 5 is displayed as it is the half.
Parameters:
Name Type Description
x number the x value of the slider.
min number the zero position of the knob.
max number the maximum postion of the knob.
factor number the scaling factor depending on the maximum values.
