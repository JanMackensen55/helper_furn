Class: EventDispatcher

EventDispatcher

This class implements a global event emitter to track the behavior of a player by recording the clicks that are made along with timestamps.

logActionbyBreak(action, card, breakValue, elapsedTime, numOfLog)

Logs an action that is associated with a pause. In other words, actions in which the subject has selected a slider, but has not moved it.
